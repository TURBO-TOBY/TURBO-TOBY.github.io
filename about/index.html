本站说明

为什么写

如果说人都有表达欲，那以写作抒发，从来都是释放表达欲的好选择。我们每天都在社交，和别人交流也是在表达，往往碎片化；每天也会思考大大小小的问题，这是内心在表达，由于没人打断，往往系统化。我想，重要话题的思考，写下来，留痕，能清晰地整理想法，也能反思自身的不足。甚至回看从前的记录，也能再次发现问题。

写在哪

然后就是写在什么平台上的问题。当然一定是数字化的平台，不可能笨拙的用纸笔记录，其最大短板在于不具备数据防丢能力。写下来当然欢迎大家看，我不会写私密性强的内容，说实在的，一切不可告人的东西，我不会以任何方式记录下来，所以要写在网络平台。前些年我曾经写到了公众号，但后来就注销了，原因就是因为没有满足这样一个很重要的要求：自由性。

国内各大社交平台对发布内容的审核，有时候是毫不讲道理，仅政治敏感一条，就有无数的理由封号、限制内容发布。敏感词库一天比一天大，因为敏感词只会增多不会减少。我不想与平台斗智斗勇，也不想担封号风险。那怎么办呢？无论是借助第三方平台还是自建服务器，只要是在国内就必须备案，必然要接受内容审查，如果发在国外网站，那你访问本站还得用网络代理，对读者来说太折腾。还好 GitHub 是少有的国内可以直接访问（即使有时候慢甚至不能访问）的网站，本站依托 GitHub 而挂载，是静态网站，免了任何内容审查，也免了实名上网的安全性问题。

为什么没评论功能

本站没有评论和留言系统，因为评论服务挂载在哪仍然有一样的问题，挂载到国外，则对大多数人不方便；挂载到国内，要么借助各大云平台托管数据，要么自建服务器，这又必然要实名上网。且主页留有联系方式（当然都是国外的），这样评论功能也不是非实现不可了。
